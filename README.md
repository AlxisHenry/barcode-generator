## How to use it ?

First, you need to clone the repository.

```bash
$ git clone https://github.com/AlxisHenry/barcode-generator
$ cd barcode-generator
```

### Install php dependencies

```
$ composer install
```

### Launch the application

```bash
php -S localhost:8080
```
